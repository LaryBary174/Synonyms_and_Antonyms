# Синонимы и Антонимы

## Описание
Это приложение для получения синонимов и антонимов для заданного слова. С использованием ИИ GigaChat

## Установка
1. Клонируем репозиторий
2. Устанавливаем зависимости
```
uv sync
```
3. Переменные окружения
- Создаем в корне проекта файл .env
- AI_CREDENTIALS = "your_ai_api_key"
- получить его возможно по [инструкции](https://developers.sber.ru/studio/workspaces/)

4. Swagger
    - http://localhost:8000/docs
    - http://localhost:8000/redoc
